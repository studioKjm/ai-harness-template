# /bdd-feature — BDD Feature Management

관련 시나리오를 Feature 단위로 묶어 관리합니다.

## Usage

```
/bdd-feature new <name> [--description|-d <desc>]
/bdd-feature list
/bdd-feature show <FTR-ID>
```

## Feature 작성 템플릿

```
In order to <비즈니스 가치>
As a <역할/사용자 유형>
I want to <기능/행동>
```

예시:
```
In order to 상품을 편리하게 구매하기 위해
As a 쇼핑몰 고객
I want to 장바구니에 상품을 담고 싶다
```
